# Huellas Solidarias - Proyecto de Rescate Animal

Huellas Solidarias es un proyecto web dedicado al rescate y adopción de mascotas. Permite registrar animales rescatados, mostrar su información y facilitar la adopción a través de una interfaz sencilla y amigable.

## Tecnologías

- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Base de datos:** MySQL  
- **Otros:** Fetch API para comunicación asincrónica

## Características

- **Registro de mascotas rescatadas:** Agrega nuevas mascotas con nombre, especie, sexo, estado, fecha de rescate y lugar.  
- **Gestión de adopciones:** Cambia el estado de la mascota (Rescatado, En tratamiento, En adopción, Adoptado).  
- **Catálogo de mascotas:** Visualiza todas las mascotas disponibles para adopción con su información y foto.  
- **Búsqueda y filtros:** Filtra mascotas por especie, sexo o estado de adopción.  
- **Interfaz responsiva:** Diseño adaptable a móviles, tablets y computadoras gracias a Bootstrap.  
- **Formulario de contacto:** Permite que los interesados en adoptar puedan comunicarse fácilmente.  
- **Comunicación asincrónica:** Uso de Fetch API para actualizar información sin recargar la página.  
- **Registro y gestión de personas:** Guarda información de las personas que rescatan o adoptan mascotas.  
- **Historial de rescates:** Mantiene un registro completo de todas las mascotas rescatadas y su evolución.
